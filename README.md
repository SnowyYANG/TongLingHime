# 通灵姬
AVG游戏，分级年龄段17+。  
注意:本游戏可能不适合患有抑郁症等精神疾病的玩家。  

2013年——  
已经整整一个月了，今天又梦见了夕阳...  
脑子痛得不行。  
不知道为什么会这样反复做着相同的梦，直到4月底，那个人去世了，之后再也没有梦见夕阳。  

1911年——  
我在反复的担心，反复的担心。  
担心公开场合这样批评一位要人会不会给我带来麻烦，导致我们不能结婚了。  
你向我保证不会有问题，无论发生什么，你都会和我结婚。  
但我们还是太天真，要结婚，人必须活着，“屈笔天诛，直笔人诛”，什么都不写才是最好的。
